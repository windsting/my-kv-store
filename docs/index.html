Limitations

The default configuration depends on free online resources which contains the following limitations:

  • Database is limited by V2 FreeDB Free Tier

    • Only one database option: MySQL
    • 50MB of database space
    • MAX QUERIES PER HOUR 800
    • MAX UPDATES PER HOUR 800
    • MAX_CONNECTIONS_PER_HOUR 800
    • MAX Connections 800
    • Any other limitations mentioned in their FAQ page.

  • Web hosting is limited by Hobby plan of Vercel

    Please refer to Limits page on Vercel official website
